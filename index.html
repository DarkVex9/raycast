WASD -- Movement

Left & Right Arrows -- Look left and right

O -- Open custom map data from the textbox

P -- Print the map data to the textbox

Escape -- Stop/start the main program loop (saves processing power)

Map Editor

There is not yet any gameplay elements, just the rendering engine and component framework. (This means no collisions with walls)

There is support for portals but they are not yet supported by the map editor. This might change in the future.

In the top down map, cells with multiple textures are shown with the NS texture on the left and the EW texture on the right.

The current version will be left as is for future reference, but a link to future versions will be added here should a future version be created.
